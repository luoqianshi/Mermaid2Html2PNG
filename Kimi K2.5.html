架构说明

前端技术栈

  • 微信小程序原生框架 / uni-app 跨端开发
  • 模块化架构：展示层、逻辑层、适配层分离
  • 统一的 API 请求封装与错误处理机制

后端技术栈

  • Supabase BaaS 平台即服务
  • PostgreSQL 关系型数据库
  • Row Level Security (RLS) 权限管控
  • RESTful API 与实时订阅
